Etapa 01 - História dos computadores

Voltar ao topo

Primeiros Computadores

A computação comercial começou a dar seus primeiros passos com máquinas gigantescas. O famoso ENIAC, por exemplo, é considerado por muitos o 1° computador digital eletrônico de grande escala.

Internet – Como surgiu

A internet nasceu em um cenário de guerra (Guerra Fria) sob o nome de ARPANET, interligando laboratórios de pesquisa. O objetivo era garantir a comunicação mesmo se parte da rede fosse destruída.

Grandes nomes envolvidos

Muitas mentes brilhantes moldaram o que usamos hoje. Aqui estão algumas das principais:

  • Tim Berners-Lee: O criador da WWW, do HTML e do HTTP.
  • Alan Turing: O pai da computação moderna.
  • Ada Lovelace: A primeira programadora da história.

Etapa 02 – O que são Clients?

Voltar ao topo

Conceito de Client no “linguajar” web

No modelo de arquitetura da internet, o Client (ou cliente) é o dispositivo ou aplicação que faz requisições para um servidor. É a ponta final que o usuário interage.

Falando sobre navegadores

Os navegadores (browsers) são os maiores exemplos de clients. Eles interpretam o código HTML, CSS e JavaScript e transformam isso em uma interface visual amigável.

Aplicações Web e Dispositivos Móveis

Hoje, as aplicações vão muito além do desktop. Com o avanço dos smartphones, o acesso mobile superou os computadores tradicionais, exigindo sites totalmente responsivos.

Etapa 03 – O que são Servers?

Voltar ao topo

Sobre servidores

Se o client pede, o Server (servidor) entrega. Eles são computadores potentes configurados para responder a requisições vindas da rede 24 horas por dia, 7 dias por semana.

Sobre hardwares e softwares em servidores

Um servidor precisa de hardware robusto (muita memória RAM e processadores dedicados) e softwares específicos para rodar serviços de banco de dados, arquivos ou páginas web (como Apache ou Nginx).

Tipos de servidores e Hospedagem de sites

Para colocar um site no ar, precisamos de um serviço de hospedagem. Existem vários tipos de servidores para isso:

  1. Servidor Compartilhado (vários sites dividindo o mesmo espaço).
  2. VPS (Servidor Virtual Privado).
  3. Servidor Dedicado (uma máquina inteira para apenas um cliente).

Etapa 04 – Linguagens de Programação

Voltar ao topo

O que é linguagem de programação

É um conjunto de instruções e regras sintáticas utilizado para dar comandos ao computador e criar softwares.

Linguagem Server-side vs Client-side

As linguagens que rodam no servidor são chamadas de Server-side (como Node.js, PHP, Python). Já as que rodam direto no navegador do usuário são as Client-side (basicamente o JavaScript).

Etapa 05 - Afinal, HTML é linguagem de programação?

Voltar ao topo

Essa é a maior polêmica entre iniciantes! HTML é uma linguagem de programação? Sim Não!

O HTML é uma linguagem de marcação de texto. Quem dá a lógica para as páginas é o JavaScript.

Etapa 06 – Entendendo a diferença entre listas ordenadas e não ordenadas

Voltar ao topo

Listas Ordenadas

As listas ordenadas são aquelas que apresentam uma sequência numérica ou alfabética. Elas são criadas usando a tag <ol> e cada item é representado por <li>.

Listas Não Ordenadas

As listas não ordenadas, por outro lado, não possuem uma sequência específica e são criadas usando a tag <ul>. Os itens também são representados por <li>, mas são marcados com bullets (pontos).
